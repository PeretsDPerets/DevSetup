# Visual Studio Code - Mes notes

## Présentation

Visual Studio Code (VS Code) est un éditeur de code gratuit développé par Microsoft.

Il est léger, rapide et extensible grâce à un grand nombre d'extensions.

---

# Installation

Téléchargement :

https://code.visualstudio.com/

---

# Extensions recommandées

## Développement

- ESLint
- Prettier
- GitLens
- Error Lens
- Path Intellisense

---

## Markdown

- Markdown All in One
- Markdown Preview Enhanced

---

## Web

- Auto Rename Tag
- Auto Close Tag
- Live Server

---

# Raccourcis utiles

Ouvrir un fichier

```
Ctrl + P
```

Recherche globale

```
Ctrl + Shift + F
```

Palette de commandes

```
Ctrl + Shift + P
```

Terminal intégré

```
Ctrl + `
```

Enregistrer

```
Ctrl + S
```

Format du document

```
Shift + Alt + F
```

---

# Terminal intégré

VS Code possède un terminal intégré.

Menu :

```
Terminal > New Terminal
```

---

# Source Control

L'onglet Source Control permet de :

- visualiser les modifications ;
- créer des commits ;
- changer de branche ;
- synchroniser avec GitHub.

---

# Paramètres utiles

Activer :

- Format On Save
- Auto Save (optionnel)
- Word Wrap
- Font Ligatures (optionnel)

---

# Bonnes pratiques

- Installer uniquement les extensions utiles.
- Mettre régulièrement VS Code à jour.
- Utiliser Prettier pour garder un code homogène.
- Organiser les fichiers dans une arborescence claire.
